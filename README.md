Проект: Домашнее задание по Технологиям Программирования - Объектно-ориентированное программирование. Тестирование. Работа с Git.
Описание: программа реализует классы Ingredient, Recipe, ShoppingList и DietaryRecipe. Данная программа помогает создавать и
хранить игредиенты с указанием самого ингредиента, количества и единицы измерения, рецепты из указанных ингредиентов, списки покупок
для указанных рецептов. В проекте реализованы тесты для проверки корректности работы программы.

Для установки необходимо выполнить следующие bash-команды:

git clone https://github.com/Pavel1Furious/tp_hw2
cd tp_hw2
pip install -r requirements.txt
pytest
